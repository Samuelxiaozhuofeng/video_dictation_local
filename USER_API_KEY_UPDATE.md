## 🎯 用户如何使用

### 步骤 1: 获取 API Key

1. 访问 [Google AI Studio](https://aistudio.google.com/app/apikey)
2. 登录 Google 账号
3. 点击 "Create API Key"
4. 复制生成的 API Key

### 步骤 2: 在应用中配置

1. 打开应用
2. 点击右上角的 **Settings**（设置）图标
3. 在 **AI Configuration** 部分找到 **"Gemini API Key"** 输入框
4. 粘贴您的 API Key
5. 点击 **Save** 保存

### 步骤 3: 开始使用

现在您可以使用 AI 功能来获取单词定义了！

## ❓ 常见问题

### Q: 我需要为每个用户提供 API Key 吗？
A: 不需要！每个用户使用自己的 Google 账号获取免费的 API Key。

### Q: 如果用户不输入 API Key 会怎样？
A: AI 功能（单词定义）将无法使用，但其他功能（视频播放、字幕显示等）仍然正常。

### Q: 这样会增加用户使用门槛吗？
A: 获取 API Key 只需要 1-2 分钟，而且是一次性操作。这是目前最安全和可持续的方案。
